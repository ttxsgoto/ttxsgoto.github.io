#### 熟练掌握下面快捷键的使用(方便查询)
---
##### Pycharm日常快捷键
```
- cmd + b           跳转到声明处(cmd + 鼠标)
- cmd + y           显示方法对应的代码
- cmd + []          光标之前/后的位置
- cmd + del         删除当前行
- cmd + o           搜索class
- cmd + l           指定行数跳转
- cmd + shift + o   搜索文件
- cmd + opt + o     搜索函数
- cmd + +           展开当前
- cmd + -           收缩当前
- cmd + shift + +   展开所有
- cmd + shift + -   收缩所有
- cmd + fn + F2     停止运行程序
- cmd + /           注释/取消注释
- cmd + fn + F12    显示文件结构
- cmd + d           在下一行复制本行的内容
- cmd + up/down     转到其他目录中
- alt + enter       自动添加包
- alt + del         向前删除一个字符串
- alt + up          选中一个单词
- alt + control + i 自动缩进行
- double shift      选择需要查看的文件
- alt + r           启动程序
- alt + d           debug启动程序
```
---

##### Vim日常快捷键
```
- hjkl              左下上右
- gg                跳到文首
- G                 跳到文末
- M                 跳到当前页屏幕中间
- '0                光标跳到上次关闭文件前的位置
- control + b       向上翻页
- control + f       向下翻页
- dw                删除一个单词
- D                 删除所在行光标后面的内容
- d0                删除所在行光标前面的内容
- x/X               向后删除一个字符，X为向前删除一个字符
- 0                 移动到当前行的第一个字符处
- $                 移动到当前行的最后一个字符处
- u                 撤销之前的操作
- control + r       向后执行刚才撤销的内容
- n                 查询时向下重复前一个查询操作
- N                 查询时向上重复前一个查询操作
- :s/old/new/g      用new替换行中所有的old
- %s/old/new/g      用new替换当前文件里所有的old
- v                 字符选择
- V                 行选择
- ctrl+v            块选择

可用于在选中的行上插入相同的内容
- 定位光标
- crtl+v           进入块编辑模式
- I                大写字母I，然后插入字符
- ESC              按两下esc 即可添加相同内容到对应的行
```

##### Items2常用快捷键
```
- cmd + t           新建标签
- cmd + w           关闭标签
- cmd + L/R         左右切换标签
- cmd + ;           查看历史命令
- cmd + r           清屏
- cmd + d           垂直分屏
- cmd + shift + d   水平分屏
- control + a       到行首
- control + e       到行末
- control + d       删除当前光标的字符
- 选中即复制 + 鼠标中键粘贴
```
